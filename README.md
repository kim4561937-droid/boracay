# 경제적 자율 시뮬레이터 (Financial Freedom Simulator) MVP

본 서비스는 사용자가 **경제적 자율 달성 시점과 달성 후 지속 가능성**을 계산하고 설계할 수 있도록 지원하는 **정적 웹(Static Web) MVP**입니다. 

---

## 1. 실행 방법

본 서비스는 외부 서버, 데이터베이스, 혹은 빌드 프레임워크가 필요 없는 **순수 정적 웹앱(Static Web App)**입니다.

1. 본 프로젝트 폴더의 루트 디렉터리에 위치한 **`index.html` 파일을 더블 클릭**하여 선호하는 브라우저(Chrome, Edge, Safari 등)에서 즉시 실행할 수 있습니다.
2. **CORS 우회 로직 탑재**: 로컬 파일 시스템 프로토콜(`file://`)로 실행할 때 브라우저 보안 정책으로 인해 로컬 CSV 데이터 로드가 거부되더라도, 어플리케이션에 내장된 문자열 데이터로 자동 전환(Fallback)되므로 아무런 장애 없이 동작합니다.

---

## 2. 주요 기능 및 특징

1. **경제적 자율 목표자산 계산**: 은퇴 시점부터 기대수명까지 필요한 순생활비를 인플레이션과 운용수익률을 반영하여 현재가치(할인값)로 계산합니다.
2. **달성 예상 나이 역산**: 현재 보유 자산, 월 저축액, 연 기대수익률을 기초로 하여 자산이 목표자산에 도달하는 예상 나이와 남은 기간을 월 단위 정밀 시뮬레이션으로 역산합니다.
3. **시나리오별 단축 효과 비교**:
   - 월 저축액 증가(+50만, +100만) 및 기대수익률 증가(+1%p, +2%p) 조건에서 목표 자산 달성 시점이 몇 년 단축되는지 일괄 계산합니다.
4. **달성 후 자산 유지 가능성 검증**:
   - 경제적 자율 달성 이후 기대수명까지의 자산 흐름을 시뮬레이션하여 자산의 지속성을 판별합니다:
     - **자산 소진형**: 기대수명 전 자산 고갈
     - **자산 유지형**: 기대수명 시점 자산이 시작 자산의 ±10% 내외로 유지
     - **자산 성장형**: 기대수명 시점 자산이 오히려 성장
5. **동적 요약 및 팁 카드**:
   - 시뮬레이션 결과를 매끄러운 한국어 문장으로 자동 조합해 요약문을 제공합니다.
   - 적립 수준과 목표 달성도에 맞춘 맞춤형 코칭 문구(응원 팁)가 표시됩니다.
6. **동적 차트 및 시뮬레이션 테이블 3종**:
   - 목표자산 달성 시뮬레이션 (현재 조건 성장선 vs 목표자산)
   - 시나리오별 달성 시점 비교 바 차트
   - 달성 후 자산 추이 라인 차트
   - 달성 전/후 자산 테이블 (토글식 전환)

## 3. 디렉터리 및 파일 구조

```text
/boracay (root)
  index.html                      # 메인 레이아웃 및 뼈대 마크업
  /assets
    /css
      styles.css                  # 다크모드 글래스모피즘 메인 스타일시트
      responsive.css              # 모바일 375px 대응 반응형 쿼리
    /js
      app.js                      # 앱 진입점 및 공유 링크 복사 파싱
      state.js                    # 입력값, 시나리오, 재계산 상태 관리
      calculator.js               # 시뮬레이션 금융 공식 구현
      formatter.js                # 한국 원화(KRW) 및 억/만원 단위 변환기
      charts.js                   # Chart.js 차트 초기화 및 업데이트 제어
      csv-loader.js               # CSV fetch 및 로컬 CORS 제한 우회 폴백 로더
      validators.js               # 나이 및 이율 범위 무결성 검증
    /data
      scenario_presets.csv        # 시나리오별 사전 로드 수익률 정보
      help_texts.csv              # 도움말 툴팁 내용
      example_inputs.csv          # 예시값 불러오기를 위한 초기 입력 데이터
    /vendor
      chart.min.js                # 로컬 오프라인용 Chart.js 라이브러리
  /docs
      LICENSES.md                 # 외부 라이브러리(Chart.js) MIT 라이선스 문서
      CALCULATION_NOTES.md        # 시뮬레이터 금융 수학 공식 세부 정의
  README.md                       # 본 문서
```
