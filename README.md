# 경제적 자유 설계 계산기 MVP

현재 월 생활비를 기준으로, **경제적 자유를 달성하는 시점에 필요한 자금**과 **그 자금을 만들기 위해 달성 전까지 매월 얼마를 적립(투자)해야 하는지**를 단리와 복리로 역산해주는 정적 웹 시뮬레이터입니다.

---

## 1. 실행 방법

본 서비스는 외부 서버, 데이터베이스, 혹은 빌드 프레임워크가 필요 없는 **순수 정적 웹앱(Static Web App)**입니다.

1. 본 프로젝트 폴더의 루트 디렉터리에 위치한 **`index.html` 파일을 더블 클릭**하여 선호하는 브라우저(Chrome, Edge, Safari 등)에서 즉시 실행할 수 있습니다.
2. **CORS 우회 로직 탑재**: 로컬 파일 시스템 프로토콜(`file://`)로 실행할 때 브라우저 보안 정책으로 인해 로컬 CSV 데이터 로드가 거부되더라도, 어플리케이션에 내장된 문자열 데이터로 자동 전환(Fallback)되므로 아무런 장애 없이 동작합니다.

---

## 2. 주요 기능 및 특징

1. **경제적 자유 달성 생활비 기준 역산**: 소득 기준이 아닌 실제 소비 수준(생활비)에서 역산합니다.
2. **인플레이션 및 물가상승 반영**: 미래의 생활비 가치를 현재 화폐 가치로 유지하지 않고 인플레이션을 누적 반영합니다.
3. **목표 달성을 위한 월 적립금 산출 (복리 vs 단리)**:
   - 복리로 투자 자산을 증식할 때의 저축액과 단리로 단순 저축할 때의 저축액 차이를 직관적으로 비교하여 복리 효과를 체감하게 합니다.
4. **경제적 자유 달성 후 자산 추이 시뮬레이션**:
   - 기대수명 시점까지 자산이 소진되거나(수익률이 낮거나 생활비가 많을 때) 자산이 계속 성장하는(수익률 수익금이 지출액을 초과할 때) 추이를 복리와 단리 두 경로로 나누어 동적 차트와 테이블로 보여줍니다.
5. **시나리오 퀵 프리셋 및 상태 공유**:
   - 보수적/기준/낙관적 시나리오를 클릭 한 번으로 불러올 수 있습니다.
   - 현재 입력 설정을 쿼리 파라미터 링크로 생성하여 공유하는 기능이 있습니다.
6. **동적 응원 및 팁 카드 제공**:
   - 산출된 투자금 규모에 맞춰 경제적 자유를 달성할 수 있도록 지지하고 코칭해주는 친근한 가이드 문구가 동적으로 갱신되어 의욕을 북돋아 줍니다.

## 3. 디렉터리 및 파일 구조

```text
/boracay (root)
  index.html                      # 메인 레이아웃 및 뼈대 마크업
  /assets
    /css
      styles.css                  # 다크모드 글래스모피즘 메인 스타일시트
      responsive.css              # 모바일 375px 대응 반응형 쿼리
    /js
      app.js                      # 앱 진입점 및 공유 링크 복사 파싱
      state.js                    # 입력값, 시나리오, 재계산 상태 관리
      calculator.js               # 역산/단복리/연도별 시뮬레이션 금융식
      formatter.js                # 한국 원화(KRW), 억/만원 단위 변환기
      charts.js                   # Chart.js 차트 초기화 및 업데이트 제어
      csv-loader.js               # CSV fetch 및 로컬 CORS 제한 우회 폴백 로더
      validators.js               # 나이 및 이율 범위 무결성 검증
    /data
      scenario_presets.csv        # 시나리오별 사전 로드 수익률 정보
      help_texts.csv              # 도움말 툴팁 내용
      yearly_return_samples.csv   # 샘플 연도별 변동 수익률 데이터
    /vendor
      chart.min.js                # 로컬 오프라인용 Chart.js 라이브러리
  /docs
      LICENSES.md                 # 외부 라이브러리(Chart.js) MIT 라이선스 문서
      CALCULATION_NOTES.md        # 시뮬레이터 금융 수학 공식 세부 정의
  README.md                       # 본 문서
```
