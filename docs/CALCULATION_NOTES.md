# 경제적 자유 설계 계산기 자금 역산 공식 명세

본 시뮬레이터에서 사용한 금융 계산 논리와 공식을 정의합니다.

## 1. 경제적 자유 달성 시점 미래 월 생활비 ($F$)
현재 월 생활비($C$)를 인플레이션율($i$)과 경제적 자유 달성까지의 잔여 기간($y_r$)을 반영하여 미래가치로 환산합니다.

$$F = C \times (1 + i)^{y_r}$$

---

## 2. 경제적 자유 달성 시점 목표 자산 ($A_{target}$)
경제적 자유 달성 후 생활 기간($y_e = \text{기대수명} - \text{달성나이}$) 동안 매년 물가상승률($i$)에 따라 증가하는 연간 생활비 지출액에서 연간 고정 연금 수입($P_{annual} = P_{monthly} \times 12$)을 차감한 **순지출액**의 현재가치(할인값) 합산으로 계산합니다. 할인율은 경제적 자유 달성 후 자산운용 수익률($r_{post}$)을 기준으로 합니다.

$$\text{연간 지출 (t년차)} = F \times 12 \times (1 + i)^{t-1}$$
$$\text{연간 순지출 (t년차)} = \max(\text{연간 지출 (t년차)} - P_{annual}, 0)$$
$$A_{target} = \sum_{t=1}^{y_e} \frac{\text{연간 순지출 (t년차)}}{(1 + r_{post})^t}$$

---

## 3. 월 필요 적립액 (복리 기준: $PMT$)
현재 자산($A_{now}$)이 경제적 자유 달성 시점까지 달성 전 연 수익률($r_{pre}$)에 따라 월 복리로 증식하는 미래 가치($FV_{assets}$)를 구한 뒤, 목표 자산과의 부족분($G$)을 매월 말 적립 조건으로 월 복리 적립식 공식을 적용해 계산합니다.

- **월 수익률**: $r = \frac{r_{pre}}{12}$
- **적립 개월 수**: $n = y_r \times 12$
- **현재 자산 미래가치**: $FV_{assets} = A_{now} \times (1 + r)^n$
- **목표 부족분**: $G = \max(A_{target} - FV_{assets}, 0)$

부족분($G > 0$)이 있을 때, 매월 적립액 $PMT$는 다음과 같습니다:

$$PMT = \frac{G \times r}{(1 + r)^n - 1} \quad (\text{단, } r > 0 \text{ 일 때})$$
$$PMT = \frac{G}{n} \quad (\text{단, } r = 0 \text{ 일 때})$$

---

## 4. 월 필요 적립액 (단리 기준: $PMT_{simple}$)
단리 교육용 비교를 위한 근사식입니다. 월 저축액이 평균적으로 전체 투자 기간의 절반 기간 동안 거치되어 복리를 적용받지 않고 단리로 이자가 발생한다는 가정 하에 적립식 미래 가치를 산정합니다.

- **현재 자산 단리 미래가치**: $FV_{simple\_assets} = A_{now} \times (1 + r_{pre} \times y_r)$
- **부족분**: $G_{simple} = \max(A_{target} - FV_{simple\_assets}, 0)$

$$PMT_{simple} = \frac{G_{simple}}{n \times \left(1 + r_{pre} \times \frac{y_r}{2}\right)}$$
