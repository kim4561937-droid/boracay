# 경제적 자율 시뮬레이터 공식 명세

본 시뮬레이터에서 사용한 금융 계산 논리와 공식을 정의합니다.

## 1. 목표 시점 실질 월 생활비 ($F$)
현재 기준 월 생활비($C$)를 인플레이션율($i$)과 경제적 자율 목표 나이까지의 잔여 기간($y_{target} = \text{목표 나이} - \text{현재 나이}$)을 반영하여 미래가치로 환산합니다.

$$F = C \times (1 + i)^{y_{target}}$$

---

## 2. 경제적 자율 목표 자산 ($A_{target}$)
경제적 자율 달성 후 기대 여명($y_{post} = \text{기대수명} - \text{목표 나이}$) 동안 매년 물가상승률($i$)에 따라 증가하는 연간 생활비 지출액을 은퇴 후 연간 자산운용 수익률($r_{post}$)로 할인하여 은퇴 시점의 현재가치로 합산합니다.

$$\text{연간 지출 (t년차)} = F \times 12 \times (1 + i)^{t-1}$$
$$A_{target} = \sum_{t=1}^{y_{post}} \frac{\text{연간 지출 (t년차)}}{(1 + r_{post})^t}$$

---

## 3. 현재 조건 기준 예상 달성 나이 및 준비 기간
현재 자산($A_{now}$)에서 매월 저축액($S$)을 달성 전 연간 기대수익률($r_{pre}$)의 월복리 수익률($r = \frac{r_{pre}}{12}$)로 굴려가며 자산의 누적 성장을 모뮬레이션합니다.

- **자산 성장 점화식**:
  $$A_{m} = A_{m-1} \times (1 + r) + S$$
- 이 시뮬레이션을 월 단위로 반복하여 처음으로 $A_{m} \ge A_{target}$을 만족하는 개월수 $m$을 구합니다.
- **예상 달성 나이**: $Age_{achieved} = \text{현재 나이} + \frac{m}{12}$
- **남은 준비 기간**: $Years_{remaining} = \frac{m}{12}$년

---

## 4. 목표 나이 도달을 위한 최소 월 저축액 ($S_{min}$)
현재 자산($A_{now}$)이 목표 나이 시점까지 월 복리로 증식하는 미래 가치($FV_{assets}$)를 뺀 부족분($G$)에 대하여 목표 기간 내에 채우기 위한 매월 최소 저축액입니다.

- **적립 개월수**: $n = y_{target} \times 12$
- **현재 자산 미래가치**: $FV_{assets} = A_{now} \times (1 + r)^n$
- **목표 부족분**: $G = \max(A_{target} - FV_{assets}, 0)$

부족분이 있을 때 매월 최소 저축액 $S_{min}$은 다음과 같습니다:
$$S_{min} = \frac{G \times r}{(1 + r)^n - 1} \quad (r > 0 \text{ 일 때})$$
$$S_{min} = \frac{G}{n} \quad (r = 0 \text{ 일 때})$$

---

## 5. 은퇴 후 자산 유지 가능성 시뮬레이션 및 판정
실제 경제적 자율을 달성한 나이와 시점의 자산($A_{start}$)을 기준으로 기대수명까지 매년 자산 운용 수익과 생활비 지출을 정산합니다.

- **연도별 정산 점화식** (재투자 ON 기준):
  $$\text{연말 자산} = \text{연초 자산} + \text{연간 수익} - \text{연간 생활비}$$
  $$\text{endingAsset}_{t} = \text{startingAsset}_{t} \times (1 + r_{post}) - \text{Expense}_{t}$$
- **재투자 OFF 기준**:
  자산 운용 수익이 연간 생활비를 초과하는 경우 초과 분은 재투자되지 않고 소멸(소비)됩니다:
  $$\text{endingAsset}_{t} = \text{startingAsset}_{t} + \min(0, \text{startingAsset}_{t} \times r_{post} - \text{Expense}_{t})$$

### 자산 상태 판정 기준
1. **자산 소진형**: 기대수명 이전에 자산이 0 이하로 떨어지는 경우.
2. **자산 유지형**: 기대수명 시점 자산이 시작 자산의 $\pm 10\%$ 범위 이내인 경우.
3. **자산 성장형**: 기대수명 시점 자산이 시작 자산보다 $10\%$ 초과하여 성장한 경우.
